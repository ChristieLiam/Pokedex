# Professor's Pokedex Web App

A dynamic and responsive web application that allows users to search for Pokémon, view detailed statistics, and save their favorites. Built entirely with Vanilla JavaScript and consuming the public PokéAPI.

## Features

### Search by Name/ID
- Find any Pokémon using its official name or Pokédex number

### Filter by Type
- View all Pokémon belonging to a specific type (e.g., "Fire", "Water")  
- Access the Favorites list through the type filter

### Detailed View
- Click any Pokémon card to see an expanded view  
- Displays stats, abilities, moves, and evolution chains

### Favorites System
- Save favorite Pokémon to a local list for quick retrieval

### Dynamic UI
- Fully built with Vanilla JavaScript  
- Uses async/await, fetch, and complex DOM manipulation

## How to Run Locally

### Step 1: Clone the repository

    git clone https://github.com/ChristieLiam/Pokedex.git

### Step 2: Navigate to the project folder

    cd Pokedex

### Step 3: Open the app

Open `index.html` in your web browser.  
No server is required.
